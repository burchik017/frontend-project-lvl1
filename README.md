## Brain-games
- brain-even: "Parity check"
- brain-calc: "Calculator"
- brain-gcd: "Greatest common factor"
- brain-progression: "Arithmetic progression"
- brain-prime: "Is it a prime number?"

#### Install:
* nodejs https://nodejs.org/en/
* clone 

##### Run:
    $ make install
    $ make link
    $ brain-games
           -even
           -calc
           -gcd
           -progression
           -prime
